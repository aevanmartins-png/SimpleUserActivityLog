# UserActivityLog
A simple C# program for tracking user activity and generating human-readable status strings.

Example: UserX was active 4 days ago

Example 2: UserY was active 1 month ago

This project provides a class to store usernames and their last active timestamps. It converts the time elapsed since their last action into a natural language string

Automatically scales from minutes to years based on how much time has passed.

Supports both Earth time and Mars time
